CARA MENAMBAHKAN LOGO:

1. Salin file gambar logo Anda (format PNG atau SVG)
2. Tempelkan di folder ini (public/)
3. Rename file menjadi: logo.svg (untuk format SVG) atau logo.png (untuk format PNG)
4. Refresh halaman browser

Logo akan muncul otomatis tanpa bingkai biru.

Format yang didukung:
- SVG (disarankan - format yang sedang digunakan, scalable)
- PNG (alternatif, raster format)

Ukuran yang disarankan: 200x150 pixels atau rasio 4:3

CATATAN:
- Sistem saat ini menggunakan logo.svg sebagai logo utama
- Jika SVG gagal dimuat, akan otomatis fallback ke logo.png
- Jika kedua file gagal, akan menampilkan logo teks "LABABIL solution"
- Pastikan file logo tidak memiliki background atau border yang tidak diinginkan

UNTUK TESTING:
- Buka http://localhost:3000/test-logo.html untuk test loading logo
- Buka http://localhost:3000 untuk melihat aplikasi utama
